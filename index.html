SmartArea

Autocomplete, dropdowns, mentions and other goodies to html textareas

Download View on GitHub

What?

smart-area is an AngularJS module that allows you to easily add huge interactivity to your textareas.

Features

  • Word highlighting (syntax, keywords... ) with support for multiple sets, and regexp
  • Autocomplete
  • Dropdown suggestions (mention users after typing @, hashtag support, keywords... )
  • Minimal configuration required, but highly customizable if necessary

Flexibility

All the features are opt-in via configuration, so you get exactly what you want. If it's not there yet, just drop an issue!
